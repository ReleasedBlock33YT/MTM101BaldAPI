** ProceduralTextureBuilder **
T:MTM101BaldAPI.AssetTools.ProceduralTextureBuilder
	-> A builder for procedural textures, AKA textures generated with other textures.
	   **This is slow! Please avoid using this for real time applications, preferably use when loading your mod!**

M:MTM101BaldAPI.AssetTools.ProceduralTextureBuilder.GetAllTexturesWithTag(System.String)
	-> Gets all currently stored textures with the specified tag.
	   **Do not use this to get your finished textures! Use ExportAll for that!**
	-> PARAMS:
		-> tag

M:MTM101BaldAPI.AssetTools.ProceduralTextureBuilder.Clip(System.String,UnityEngine.Texture2D,System.String)
	-> Clip the texture using all textures in the tag as masks, and put the resulting textures into the specified tag.
	-> PARAMS:
		-> tag
		-> toClip
		-> resultTag

M:MTM101BaldAPI.AssetTools.ProceduralTextureBuilder.SplitViaColor(UnityEngine.Texture2D,System.String)
	-> Splits the specified texture into multiple textures via color, and puts all the resulting textures into the specified tag.
	   **Only use this in specially designed images with few colors!**
	-> PARAMS:
		-> toSplit
		-> tag

M:MTM101BaldAPI.AssetTools.ProceduralTextureBuilder.Trim(System.String,System.String)
	-> Trims all the textures so that their bounds are only where there are visible pixels.
	-> PARAMS:
		-> tag
		-> resultTag
